# Animations de Pose Assise - War Room

Ce dossier contient les **animations idle** (en boucle) pour les personnages assis à la War Room.

## ⚠️ Important

Ce sont des animations **déjà assis** (idle/loop), PAS des animations de transition !

### Ordre des animations dans la War Room :

1. `Stand Up.fbx` - Se lever (transition)
2. `Mr_Man_Walking.fbx` - Marcher (loop)
3. `Stand To Sit.fbx` - S'asseoir (transition) ← **Toujours celle-là**
4. **Votre animation** → Pose assise personnalisée (loop)

## Fichiers attendus

Des animations **en boucle** de personnages **déjà assis** :

- `Sit Down.fbx` - Assise standard
- `Sit Relaxed.fbx` - Détendu 
- `Sit Formal.fbx` - Dos droit, formel
- `Sit Thinking.fbx` - Main sur le menton
- `Sit Crossed Arms.fbx` - Bras croisés
- `Sit Legs Crossed.fbx` - Jambes croisées
- etc.

## Format requis

- **Type** : Animation idle (boucle)
- **Format** : `.fbx`
- **État** : Le personnage doit être **déjà assis** dans la pose
- **Timing** : Animation qui boucle (LoopRepeat)

## Si pas d'animations

Si le dossier est vide, le personnage restera figé dans la pose finale de "Stand To Sit".
